# Lung Nodule Segmentation using U-Net

## Overview
This project implements a U-Net based deep learning model for lung nodule segmentation from CT scan images.

## Tech Stack
- Python
- TensorFlow
- OpenCV
- NumPy

## Features
- U-Net Architecture
- Dice Loss & Dice Coefficient
- Image Preprocessing
- Data Augmentation
- Segmentation Mask Generation
- Prediction Visualization
